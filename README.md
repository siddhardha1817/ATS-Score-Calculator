# ATS Score Calculator

An AI-powered ATS (Applicant Tracking System) Resume Analyzer that evaluates resumes against job descriptions and provides ATS compatibility scores along with improvement suggestions.

---

## Features

- Upload resume in PDF or DOCX format
- Analyze resume against job description
- ATS score calculation
- Keyword matching analysis
- Resume improvement suggestions
- Clean and responsive UI
- Fast and simple workflow

## Installation

### Clone the Repository

```bash
git clone https://github.com/siddhardha1817/ATS-Score-Calculator.git
```

### Navigate to the Project Folder

```bash
cd ATS-Score-Calculator
```

### Install Required Dependencies

```bash
pip install -r requirements.txt
```

### Run the Flask Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```

---

## How It Works

1. Upload your resume file
2. Paste the job description
3. Click the Analyze button
4. The system compares resume keywords with job requirements
5. ATS score and suggestions are generated
